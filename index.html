Version 1.0.0   ·   GitHub Project   ·   Created by Studio Junkyard
Born out of frustration of other Font Icon Sets not containing Icons needed when developing LibreCMS, we created LibreICONS to fill the gap that we needed, and ended up consolidating all the Font Icons we were using into one set.

Then we decided to share it with you.

Not only did we include our own CSS Styles .libre .libre-cannabis, we also included the references and styling extra's and references from the Font Awesome, and Glyphicons projects to help making the transition easier. As well, as adding the CSS Animation, including other extra's.
