The Open Source Icon Web Font and CSS toolkit

Version 2.0.0   ·   GitHub Project   ·   Created by Studio Junkyard
Born out of frustration of other Font Icon Sets not containing Icons needed when developing LibreCMS, we created LibreICONS to fill the gap that we needed, and ended up consolidating all the Font Icons we were using into one set.

Then we decided to share it with you.

And now with Version 2.0.0 we've given the option of using the Icons as SVG, either as SVG Files, or you can use them as CSS References much like how Font Icons work.

Not only did we include our own CSS Styles .libre .libre-activity, we also included the references and styling extra's and references from the Font Awesome, and Glyphicons projects to help making the transition easier.
