Introduction

The most important thing to note about the below examples, is that we've only added extra CSS Classes, we haven't added any new jQuery or Javascript to add extra functionality. The below components that do use jQuery or Javascript is what it included with Bootstraps Default libraries.

Basic Icons

libre-tardis

You can place LibreICONS icons just about anywhere using the CSS Prefix libre and the icon's name. LibreICONS is designed to be used with inline elements (we like the <i> tag for brevity, but using a <span> is more semantically correct). 

<i class="libre libre-tardis"></i> libre-tardis
If you change the font-size of the icon's container, the icon gets bigger. Same things goes for color, drop shadow, and anything else that gets inherited using CSS.

To increase icon sizes relative to their container, use the libre-lg (33% increase), libre-2x, libre-3x, libre-4x, or libre-5x classes.

<i class="libre libre-tardis libre-2x"></i> libre-2x
<i class="libre libre-tardis libre-3x"></i> libre-3x
<i class="libre libre-tardis libre-4x"></i> libre-4x
<i class="libre libre-tardis libre-5x"></i> libre-5x
If your icons are getting chopped off on top and bottom, make sure you have sufficient line-height.

Colored (or "Coloured") Icons

Icon color's (or "Colours") can be changed using color-*, with * being the colour of choice. These choices are classed similar and use the palette from Bootstrap 3's default palette. You can also use Bootstrap's text-* colour options. Obviously you can change the colour's via CSS. The "Normal" Colour of icons is determined by the colour set in the site's CSS.

Stacked Icons

libre-social-facebook on libre-square-rounded
libre-reddit on libre-circle
libre-code on libre-square
libre-ban on libre-user

To stack multiple icons, use the libre-stack class on the parent, the libre-stack-1x for the regular sized icon, and libre-stack-2x for the larger icon. text-inverse can be used as an alternative icon color. You can even throw larger icon classes on the parent to get further control of sizing.
