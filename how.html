Using the Icons:

  1. After downloading the project, copy the libreicons.css file from the project css/ folder, and the font/ folder directory into your project.
  2. In the <head> of your html, reference the location to your libreicons.css. 
    <link rel="stylesheet" href="path/to/libreicons/css/libreicons.css">
  3. To use the icons in your pages you use 
    <i class="libre libre-activity"></i>
  4. Check out the examples and start using LibreICONS!
