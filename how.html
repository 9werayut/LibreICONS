  How

How to get LibreICONS 1.0.5 into your site!

Setting up LibreICONS is as easy as adding a <link rel="stylesheet" href="path/to/your/css/libreicons.css"> in your <head> area of your page/'s.

EASY: Default CSS

Use this method to get the default LibreICONS CSS.

  1. After downloading the project, copy the libreicons.css file from the project css/ folder, and the font/ folder directory into your project.
  2. In the <head> of your html, reference the location to your libreicons.min.css. 
    <link rel="stylesheet" href="path/to/libreicons/css/libreicons.css">
  3. Check out the examples to start using LibreICONS!
