  How

How to get LibreICONS 2.0.0 into your site!

Using the Font Icons:

  1. After downloading the project, copy the libreicons.css file from the project css/ folder, and the font/ folder directory into your project.
  2. In the <head> of your html, reference the location to your libreicons.css. 
    <link rel="stylesheet" href="path/to/libreicons/css/libreicons.css">
  3. To use the icons in your pages you use 
    <i class="libre libre-activity"></i>
  4. Check out the examples and start using LibreICONS!

Using the SVG Icons:

There are at this stage two ways to use the SVG Icons

As SVG Images

  1. As above, after downloading the project, copy the libreicons-svg.css files from the css/ folder, and the svg/ folder directory into your project.
  2. In the <head> of your html, reference the location to your libreicons-svg.css. 
    <link rel="stylesheet" href="path/to/libreicons/css/libreicons-svg.css">
  3. To use the icons in your pages you use 
    <svg class="libre"><use xlink:href="path/to/libre-activity.svg#libre-activity"></use></svg>
